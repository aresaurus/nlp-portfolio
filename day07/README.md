# Day 7 - Hangman (100 Days of Code)

Base project from Angela Yu's 100 Days of Code course, extended with:

- Spanish linguistics vocabulary (synonyms, antonyms, polysemy, pragmatics...)
- Unicode accent normalization — guessing "o" reveals both "o" and "ó"
- "ñ" handled as a distinct character
- Definition displayed at the end of each game

Built with: Python · unicodedata
