# READ THE ROOM 🎮

A Python terminal game inspired by the classic Rock, Paper, Scissors, with a sociolinguistic twist.

## About
In linguistics, **register** refers to the variety of language used in a particular social situation. A competent speaker doesn't just know the language — they know *when* to use each register. This game puts that to the test.

Each round presents a random social scenario. Your job: pick the right register (**Formal**, **Informal** or **Jargon**) before you embarrass yourself.

## How to play
1. Run `read_the_room.py`
2. Read the situation carefully
3. Type `Formal`, `Informal` or `Jargon`
4. Find out if you read the room correctly

## Built with
- Python 3
- `random` module
